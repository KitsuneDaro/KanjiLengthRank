# これは何？
漢字の線の長さを求めてランキング化します

# 必要なもの
## KanjiVGのデータ(main)
https://kanjivg.tagaini.net/

# ソースコードの説明
## kanjivg_get_length.py
漢字の線の長さを求めます
使う際にはソースコード中KanjiVGのパスを環境に合わせて変更してください

## kanjivg_get_length_fix.py
kanjivg_get_length.pyのデータを修正します

## analyze.ipynb
漢字の線の長さ(length)のランキングを出します
画数(count)のランキングも出せます
